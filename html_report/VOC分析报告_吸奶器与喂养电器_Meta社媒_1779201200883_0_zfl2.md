
### 0.4 数据局限性（必读）

1. **品牌官方内容，非用户评论**：本数据为品牌在 FB/IG 官方账号发布的内容及其互动数据，VOC 部分基于内容文字中的痛点描述和正向表达，并非独立用户评论数据。若需真正的用户 VOC 分析，建议补充评论区数据。
2. **视频类数据缺乏文本**：FB 视频文件（432条）几乎无文字描述，无法参与文本标注。
3. **"喂养电器-未细分"主要是 Registry/Essentials 场景帖**：并非严格意义的喂养电器产品帖，分析时需注意。
4. **消毒器与暖奶器样本量较小**（去重后各约 30 条），统计结论具方向性参考价值，建议持续积累数据验证。

## 四、产品连带率与交叉兴趣分析

### 4.1 连带率数据

| 连带组合 | 共同提及数 | 连带率（以分子品类为基准） |
|---|---|---|
| 吸奶器 → 喂养电器 | 149 | 6.0%（吸奶器帖中有喂养电器） |
| 吸奶器 → 消毒器 | 128 | 5.2%（吸奶器帖中有消毒器） |
| 吸奶器 → 暖奶器 | 7 | 0.3%（吸奶器帖中有暖奶器） |
| **消毒器 → 吸奶器** | **128** | **61.0%**（消毒器帖中有吸奶器） |
| 暖奶器 → 吸奶器 | 7 | 7.3%（暖奶器帖中有吸奶器） |
| 消毒器 ↔ 暖奶器 | 1 | <1% |

> **核心洞察**：
> - **消毒器与吸奶器的连带极强（61%）**：超 6 成消毒器帖会同时提到吸奶器，说明两者在内容体系中已形成自然搭配（nursing essentials / 泵奶清单语境）
> - **暖奶器与吸奶器连带极低（0.3%/7.3%）**：两类产品在内容层面几乎完全分割
> - **消毒器与暖奶器之间几乎无连带**：两个子品类目前各自为战，缺乏"喂养电器整体方案"叙事

### 4.2 场景链路现状与机会

**已存在链路**：泵奶 → 奶瓶清洁消毒（吸奶器 + 消毒器联动，内容已有验证）

**待建立链路**：泵奶 → 储奶 → **暖奶**（暖奶器独立碎片化，认知路径缺失）

**完整机会链路**（当前缺失中间环节）：

```
吸奶（M5/Air1）→ 储奶（储奶袋/保冷包）→ 暖奶（Milk Warmer）→ 消毒（KleanPal Pro）→ 喂养（奶瓶）
```
