- **报告编写规范**：每个模块末尾必须产出【结论句】；不用模糊措辞，无数据支撑需标注【推断：需XX数据验证】；新增图表遵循Fortune Red色卡系统；ECharts遵循防溢出规范；新增chart div对应initChart函数并在initAllCharts中用safeInit包裹
